# cryptoUnicornsBot
A Crypto Unicorns Discord bot. Track the shadowcorn act 1 related activites, UNIM and RBW price, and calculate the breeding costs.

==============================

Oauth Discord scopes:
- applications.commands
- bot
  - send messages

==============================

Active commands:
- /unimleft
- /unimprice
- /rbwprice
- /breedunicorn
