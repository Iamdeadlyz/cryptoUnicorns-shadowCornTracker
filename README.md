# cryptoUnicornsBot
A Crypto Unicorns Discord bot. Track the remaining shadowcorn eggs, lootbox, UNIM left in the dark forest, and the price.

==============================

Oauth Discord scopes:
- applications.commands
- bot
  - send messages

==============================

Commands:
- /shadowcorn
- /lootbox
- /unimleft
- /unimprice

==============================

- Track the remaining shadowcorn eggs
